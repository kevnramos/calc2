Getting Started with Docker

Basic intro to Docker

Showing all your containers

There are a multitude of different docker commands that you can use

You can run:

docker ps -a

to show all of the containers you currently have


Downloading an existing container

You can download containers from Docker Hub

We can run ubuntu from a docker container

To do so, we can simply run:

docker pull ubunutu

This will download the LTS version of ubunutu (currently v20.04)


Running the ubunutu image

To run the container that has ubuntu on it, we first need the name of the container

To get the name, we can do:

docker ps -a

At this point you should see a list of containers, most likely only one will appear because this is the first time you use docker

run:

docker start <container_name>

Now that the container is started, we have to run it in order to get into the ubuntu shell

docker run -it <container_IMAGE_ID>

you can get the IMAGE_ID from

docker ps

you should now be inside a full fledged ubunutu container Running directly on your terminal
